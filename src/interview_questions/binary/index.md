# 二进制运算

### 二进制加法

以0111+1110为例子. 

- 1:先把两个数对齐，和十进制计算时候第一步一样。  
  ![图1](./assets/二进制/1.png)
- 2:从最右边对齐的上下两个数开始。  
  ![图2](./assets/二进制/2.png)
- 3:1+0=1，0+0=0.如果两个数不是都是1，那么直接相加就好。直接写在下面. 
  ![图3](./assets/二进制/3.png)
- 4:1+1=10，把1写上面，把0写下面。  
  ![图4](./assets/二进制/4.png)
- 5:1 +1 +1 = 11，要注意进位。  
  ![图5](./assets/二进制/5.png)
- 6:依次类推，计算完成。  
  ![图6](./assets/二进制/6.png)
