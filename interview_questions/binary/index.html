# 二进制加法

以0111+1110为例子.

  • 1:先把两个数对齐，和十进制计算时候第一步一样。
  • 2:从最右边对齐的上下两个数开始。
  • 3:1+0=1，0+0=0.如果两个数不是都是1，那么直接相加就好。直接写在下面.
  • 4:1+1=10，把1写上面，把0写下面。
  • 5:1 +1 +1 = 11，要注意进位。
  • 6:依次类推，计算完成。
